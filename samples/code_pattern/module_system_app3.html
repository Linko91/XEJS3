Per eseguire il codice ES6 su qualsiasi Browser ho usato Typescript 1.5+ alpha:

npm  install  -g  typescript
  • Ho rinominato tutti i file ES6 in .ts
  • Ho creato il file app/tsconfig.json con il seguente contenuto: 
    {
    "compilerOptions": {
        "module": "amd",        
        "sourceMap": true
	},
    "files": [
        "lib/math.ts",
        "lib/mathplus.ts",
        "lib/mathplusplus.ts",
		"app1.ts",
		"app2.ts",
		"app3.ts"
    ]
}
  • Ho compilato (transpile da ES6 --> ES5) tutto il codice in modo da ottenere puro JS utilizzabile su qualsiasi Browser! 
    tsc  --project  app  --target  ES5
  • In fine siccome ho specificato AMD come tipo di module nella compilazione, nella pagina HTML basta incluedere lo script di RequireJS (scaricato da: http://requirejs.org/docs/release/2.1.17/minified/require.js) e indicare l'entrypoint (data-main) dell'applicazione: 
    <script src="require.min.js" data-main="app/app3.js">
